Capture the Flag competitions

A capture the flag competition is a computer security competition. Teams usually race to get points. They usually serve an educational purpose. There are two common types of CTFs.

Jeopardy-style CTFs involve multiple categories of problems, each usually having different point values.

In attack-defence CTFs, each team has a machine or network which they must secure against attacks. Each team then attempts to attack each other to get points.

ctftime.org is a useful resource.
