Wargames

A wargame is a security challenge where one must exploit/defend a vulnerability in a system/application or gain/prevent access to a computer system.

List of good wargames for beginners

  • bandit - teaches the basics needed to play other wargames
  • natas - a web-based wargame where you have to find the password to the next level
  • io - exploit a program to get to the next level
